Blogging Platform API

Bienvenido a la API de Blogging Platform

Endpoints principales

  • GET /api/blogs - Listar publicaciones
  • GET /api/blogs/:id - Obtener una publicación
  • POST /api/blogs - Crear una nueva publicación
  • PUT /api/blogs/:id - Actualizar una publicación
  • DELETE /api/blogs/:id - Eliminar una publicación
